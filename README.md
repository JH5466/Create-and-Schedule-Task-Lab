# Create-and-Schedule-Task-Lab

This project demonstrates how to create and schedule an automated weekly task in Windows Task Scheduler. The task was configured to run every Tuesday at 12:00 PM and launch Firefox, open a new tab, and navigate to a specific website. The lab focuses on understanding triggers, actions, and verifying successful task execution through manual testing.
